Welcome to the text demo of mircoservices in stackato

Overview

  • A simple web-application for accessing accounts and products data.
  • All the account and product information is fetched via a zuul reverse proxy to the Products and Accounts microservices.

Netflix Libraries in this demo

  • Spring Cloud Config Server for external configuration
  • Spring Cloud Eureka for service registration and discovery
  • Spring Cloud Zuul as a reverse proxy for backend microservices

The Demo

  • Fetch account #123456714: /accounts/123456714
  • Fetch product #11111111: /products/11111111
  • Fetch an account by name: /accounts/owner/Michael
  • Fetch a product by name: /products/productName/Product 1
  • Account Search
  • Product Search
